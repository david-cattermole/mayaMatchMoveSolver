mmSolver.tools.createmarker¶

Create a marker on a camera, from scratch.

Tools¶

The Create Marker tool.

mmSolver.tools.createmarker.tool.is_startup_cam(x)¶

Return True if the given camera node is a ‘startupCamera’.

A startup camera is a camera; ‘persp’, ‘side’, ‘top’, ‘front’, etc.

Return type:bool
mmSolver.tools.createmarker.tool.is_not_startup_cam(x)¶

Return True if the given camera node is NOT a ‘startupCamera’.

A startup camera is a camera; ‘persp’, ‘side’, ‘top’, ‘front’, etc.

Return type:bool
mmSolver.tools.createmarker.tool.main()¶

Create a new marker under the current viewport camera, or under the selected camera, if a camera is selected.

mmSolver.tools.createmarker.tool.create_marker()¶
